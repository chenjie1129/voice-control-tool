# 语音控制助手

## 功能特性
- 通过唤醒词"小助手"激活
- 支持中文语音指令识别
- 可执行打开应用、查询信息等操作

## 使用说明
1. 运行程序后会在菜单栏显示麦克风图标
2. 说出"小助手"后接指令，例如：
   - "小助手 打开浏览器"
   - "小助手 查看日历"
   - "小助手 关闭脚本"

## 系统要求
- macOS 10.15+
- 需要麦克风访问权限
